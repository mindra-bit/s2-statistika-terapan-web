## Isi paket

- `references_multivariat.bib`
- `brown_gradient_style.css`
- `Materi_Analisis_Multivariat_Tingkat_Lanjut_preview.html` (preview HTML statis; chunk R tidak dieksekusi)

## Cara render di RStudio
1. Buka file `.Rmd` di RStudio.
2. Pastikan file `.bib` dan `.css` berada pada folder yang sama.
3. Klik `Knit` -> `Knit to HTML`.
4. Jika beberapa paket belum ada, jalankan `install.packages(...)` pada bagian awal modul.
