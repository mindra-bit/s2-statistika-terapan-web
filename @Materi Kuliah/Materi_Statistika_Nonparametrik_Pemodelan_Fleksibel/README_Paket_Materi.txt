# Paket Materi Statistika Nonparametrik dan Pemodelan Fleksibel

Isi paket:

1. `Materi_Statistika_Nonparametrik_Pemodelan_Fleksibel.Rmd` — file utama R Markdown.
2. `nonparam_flexible_refs.bib` — database referensi BibTeX untuk sitasi.
3. `Materi_Statistika_Nonparametrik_Pemodelan_Fleksibel.html` — preview HTML statis dari Rmd.

Catatan teknis: HTML preview dibuat dengan Pandoc di lingkungan ini. Chunk R tidak dieksekusi karena R tidak tersedia pada sandbox, tetapi file Rmd sudah disiapkan agar dapat dirender di RStudio dengan `rmarkdown::render("Materi_Statistika_Nonparametrik_Pemodelan_Fleksibel.Rmd")`.
