Cara render di R/RStudio:
1. Simpan file .Rmd dan sem_references.bib dalam folder yang sama.
2. Jalankan:
   install.packages(c("rmarkdown", "knitr", "lavaan", "semPlot", "semTools", "psych", "tidyverse"))
   rmarkdown::render("Materi_Pemodelan_Persamaan_Struktural_SEM_S2_Statistika_Terapan.Rmd")
3. Jika hanya ingin membaca materi tanpa menjalankan kode, gunakan file HTML preview.
